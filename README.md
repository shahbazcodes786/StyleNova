🛍️ StyleNova

Welcome to StyleNova!

StyleNova is a Django-based online shopping web application that I am developing as my school project. This project represents my dedication, hard work, and passion for learning web development.

🎯 Project Goal

My goal is to build a modern, secure, and user-friendly e-commerce platform while improving my Django and Python skills.

InshaAllah, I hope to launch StyleNova as a real startup in the future.

🚀 Features (Planned & Current)

- User Registration & Login
- Email Verification
- Product Catalog
- Product Categories
- Shopping Cart
- Wishlist
- Secure Authentication
- Responsive Design
- Order Management
- Admin Dashboard
- Search & Filters
- Payment Gateway Integration (Future)

🛠️ Tech Stack

- Python
- Django
- HTML5
- CSS3
- JavaScript
- Bootstrap
- SQLite (Development)

📚 About This Project

This project is being built entirely by me as part of my school learning journey. Every new feature helps me improve my programming and problem-solving skills.

🌟 Vision

I believe every big company starts with a small idea. StyleNova is my first step toward building something meaningful.
